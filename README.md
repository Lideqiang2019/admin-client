# 后台管理项目

这是一个后台管理项目，主要使用react/antd/axios等搭建，本仓库将项目按照功能进行拆分，不同功能分别上传，方便查看代码变更。

## 注意

### 学习的重点
  1. 同一类功能进行比较学习，比较异同点，加深对方法的理解
  2. 尝试合并函数，用一种函数来解决增加与修改问题
  3. 掌握优化的方法，数组扩容替代重发请求、生命周期与pureComponet等
